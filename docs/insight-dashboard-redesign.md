```
┌─────────────────────────────────────────────────┬──────────────┐
│  메인 컬럼 (flex-1, min 680px)                   │ 우측 패널    │
│                                                  │ (340px)      │
│  ① KPI 히어로 (4개 카드)                         │ sticky top   │

│                                                  │ 카드 목록    │
└─────────────────────────────────────────────────┴──────────────┘
```

## 2. 섹션별 명세

### ① KPI 히어로
- 오늘 매출 / 기회손실 추정 / 폐기 비용 / 즉시 조치 건수
- 배경: 오렌지-핑크 그라디언트 소프트 (#fff7f1 → #fff → #fff4fb)
- 트렌드 배지: ▲ 전일 대비 %

### ② 즉시 조치 필요
- 긴급 액션 카드 목록
- 해당 탭으로 이동 버튼
- 에이전트 배지 (제이/로이/루나)

### ③ 지금 봐야 할 이슈
- 심각도별 왼쪽 보더 컬러 (긴급/주의/정보)
- 에이전트 배지 + 감지 시각

### ④ 생산 현황 (Agent 제이) ← 재배치
- 품목별 소진 예상 프로그레스바
- 기회손실 금액 표시
- 상태 배지: 즉시 생산 / 생산 권장 / 정상

### ⑤ 카테고리 기여도 탭 ← 강화
- 탭: 매출 기여 | 판매 수량 | 마진율 | 배달 비중 | 매장 비중
- 수평 바 차트로 카테고리별 비교
- 각 탭 클릭 시 기준값 변경 (애니메이션 전환)

### ⑥ 오늘의 핵심 체크리스트
- 루나가 트렌딩 기반 자동 생성
- 카테고리 배지: 생산/발주/매출/운영
- 체크 원형 UI (POC — 상태 변경 없음)

## 3. AI 신뢰도 패널 (우측 sticky, 신규)

> 점주의 AI 신뢰도를 높이기 위한 핵심 장치
> "루나가 왜 이렇게 판단했는지" 근거를 투명하게 공개

### 구성 요소

#### 3-1. 에이전트 확신도
- 제이 / 로이 / 루나 각각 확신도 게이지 (0-100%)
- 다크 카드 배경으로 신뢰감 표현
- 실시간 pulse 애니메이션 (활성 상태)

#### 3-2. 판단 근거 공개 (Explainable AI)
- "루나의 판단 근거" 섹션
- 근거 3가지 리스트
  - 📊 데이터 소스 (예: "최근 7일 시간대별 매출 데이터")
  - 📈 패턴 인식 (예: "월요일 오후 2-4시 반복 하락 패턴 감지")
  - 🔗 외부 요인 (예: "날씨 API — 오늘 오후 강수 확률 68%")

#### 3-3. 학습 기반 배지
- "이 판단은 강남역점 37일치 데이터 기반"
- "유사 상황 정확도 84%"
- 점주가 AI를 믿을 수 있는 수치 근거 제공

#### 3-4. 즉시 추천 액션
- 루나/제이/로이가 제안한 상위 3개 액션
- 예상 효과 퍼센트 표시
- "실행안 보기" 버튼

## 4. UX 원칙

| 원칙 | 적용 |
|------|------|
| 정보 우선순위 | 긴급 → 주의 → 정보 순서 고정 |
| Explainable AI | 판단 근거 항상 노출 (점주 신뢰도) |
| 원클릭 이동 | 이슈/조치 카드에서 해당 탭 바로 이동 |
| 실시간 감지 | pulse 애니메이션으로 라이브 데이터 표현 |
| 신호등 체계 | 긴급(검정) / 주의(오렌지) / 정보(회색) |
